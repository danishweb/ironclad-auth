# Code of conduct

We want collaboration to stay respectful and productive. Harassment or abusive behavior is not tolerated.

For concrete expectations, this project follows the **[Contributor Covenant v2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct/)** text.

**Enforcement:** contact the repository maintainers via a private channel (for example GitHub Security Advisories / maintainer email if listed). Reports will be reviewed promptly.
